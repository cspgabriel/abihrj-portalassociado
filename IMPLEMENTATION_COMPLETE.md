## 🔗 Como Acessar

### Opção 1: Começar pela Página Principal
1. Abra: **`index-benefits.html`**
2. Você terá acesso a:
   - Explicação geral do portal
   - Grid visual com os 8 benefícios
   - Vídeo de demonstração rápida (25 segundos)
   - Botões para acessar cada página individual

### Opção 2: Acessar Benefício Específico
- Clique diretamente no botão "Acessar" de qualquer benefício
- Ou acesse direto: `benefit-{nome}.html`

### Opção 3: Navegação Entre Páginas
- Use o **sidebar esquerdo** para pular para qualquer benefício
- Use os botões **Anterior/Próximo** para navegar sequencialmente
- Todos os links funcionam em ambas as direções
